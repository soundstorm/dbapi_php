# dbapi_php
Deutsche Bahn API in PHP

Flexible API für Deutsche Bahn Abfahrtstafeln und Bahnhofs-Abfragen.
Beispiel enthält eine lokal generierte Abfahrtstafel mit Verspätungsanzeige.
